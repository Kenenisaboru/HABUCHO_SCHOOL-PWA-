## 📋 Quick Start (For Future Reference)

```bash
# Backend Setup
cd server
npm install
cp .env.example .env
# Edit .env with DB credentials
npm run migrate
npm run seed
npm run dev

# Frontend Setup (New Terminal)
cd client
npm install
echo "VITE_API_URL=http://localhost:5000/api" > .env.local
npm run dev

# Open http://localhost:5173/ in browser
```
